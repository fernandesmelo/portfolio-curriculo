Sobre Mim

Atualmente, sou estudante de Desenvolvimento de Software, cursando Superior de Tecnologia em Análise e Desenvolvimento de Sistemas na Faculdade Senac Pernambuco, onde pude iniciar a Residência Tecnológica do Porto Digital. Também estou me especializando como Desenvolvedor Frontend pela DevMedia.

Na minha experiência como estagiário na Aceleradora Ágil, atuei como Desenvolvedor Web Fullstack, onde ganhei experiência prática no desenvolvimento ágil de produtos.

Também já participei de dois Programas de Extensão Tecnológica (PETs) financiados pela FACEPE: PET – Ciência de Dados para Auxiliar na Tomada de Decisões em Agendamentos de Atendimentos no CISAM, onde atuei como Analista de Dados em um projeto para o CISAM - Centro Universitário Integrado de Saúde Amaury de Medeiros e PET – Projeto Monty Python: Formação de Análise Preditiva, onde atuei como Desenvolvedor Web Backend em um projeto para o Colégio Aba Global Education.
